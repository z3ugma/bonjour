# Bonjour

Greetings in several world languages

## Installation

Add this line to your application's Gemfile:

    gem 'bonjour'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bundler

## Usage

```ruby
Bonjour.hi()
=>"hello world"

Bonjour.hi("french")
=>"bonjour, tout le monde"
```
